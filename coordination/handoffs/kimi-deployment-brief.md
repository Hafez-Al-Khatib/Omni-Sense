### Task 2 — Bootstrap server

```bash
# As root after first ssh
apt update && apt upgrade -y
curl -fsSL https://get.docker.com | sh
apt install -y docker-compose-plugin git ufw
ufw allow 22/tcp 80/tcp 443/tcp 8883/tcp
ufw --force enable
useradd -m -s /bin/bash -G docker omni
mkdir -p /opt/omni-sense && chown omni:omni /opt/omni-sense
```

### Task 3 — DNS

- Use a domain the user owns, OR free DuckDNS (https://www.duckdns.org/).
- Recommended subdomain plan if `omnisense.example.com` is the apex:
  - `eep.<domain>` → EEP API
  - `grafana.<domain>` → Grafana
  - `prometheus.<domain>` → Prometheus
  - `mlflow.<domain>` → MLflow
  - `mqtt.<domain>` → MQTT (TLS, port 8883)
- All A records → the Hetzner IP. Wait for propagation (`dig +short eep.<domain>`).

### Task 4 — Caddy reverse proxy (commit to repo)

Create `Caddyfile` at repo root:

```caddy
# Caddyfile — Omni-Sense production reverse proxy
# Deployed on the Hetzner CX22; Caddy runs as a separate container in
# docker-compose (add to docker-compose.prod.yml — see Task 5).

eep.{$OMNI_DOMAIN} {
    reverse_proxy eep:8000
    encode gzip
    request_body {
        max_size 10MB
    }
}

grafana.{$OMNI_DOMAIN} {
    reverse_proxy grafana:3000
}

prometheus.{$OMNI_DOMAIN} {
    reverse_proxy prometheus:9090
    # Basic auth (override OMNI_PROM_BASIC in .env.production)
    basicauth {
        admin {$OMNI_PROM_BASIC_HASH}
    }
}

mlflow.{$OMNI_DOMAIN} {
    reverse_proxy mlflow:5000
    basicauth {
        admin {$OMNI_MLFLOW_BASIC_HASH}
    }
}
```

Generate basic-auth hashes with `docker run --rm caddy:2 caddy hash-password --plaintext '<pwd>'`.

### Task 5 — Production compose overlay

Create `docker-compose.prod.yml` at repo root (additive overlay):

```yaml
services:
  caddy:
    image: caddy:2
    restart: unless-stopped
    ports:
      - "80:80"
      - "443:443"
    env_file: .env.production
    volumes:
      - ./Caddyfile:/etc/caddy/Caddyfile:ro
      - caddy_data:/data
      - caddy_config:/config
    networks:
      - omni-net
    depends_on:
      - eep
      - grafana
      - prometheus
      - mlflow

  # Disable host port bindings on the apps — Caddy fronts them
  eep:
    ports: []
  grafana:
    ports: []
  prometheus:
    ports: []
  mlflow:
    ports: []

volumes:
  caddy_data:
  caddy_config:
```

Deploy command:

```bash
cd /opt/omni-sense
git clone <repo url> . && git checkout feature/field-verification
cp .env.production .env  # see Task 6
docker compose -f docker-compose.yml -f docker-compose.prod.yml up -d
```
